# Arabic Sweets E-Commerce

---

## 🚀 Features

* Browse products in the shop page
* Search products by name or description
* Sort products by price (ASC/DESC)
* Add products to cart
* Update cart quantities or remove items
* View order summary with total and shipping cost
* Create an order and clear the cart

---

## 🧭 Pages

* **Home**

  * Hero section with call-to-action
  * About section
  * "Best sellers" products

* **Shop**

  * Product list
  * Search bar
  * Price sorting
  * Add to cart button

* **Cart**

  * List of selected products
  * Quantity controls
  * Order summary (total + shipping)
  * Checkout button

---

## 🧠 Order Flow

* User adds products to cart
* Cart updates in real time
* At checkout:

  * Order is created in the database
  * Total is calculated on the backend
  * Shipping cost is added
  * Cart is emptied

---

## 🛠️ Tech Stack

* Frontend: React (Vite), Bootstrap, CSS
* Backend: Node.js, Express
* Database: MySQL

---

## 📦 Setup

### Backend

```bash
cd arabic-sweets-backend
npm install
npm run start
```

or (development mode):

```bash
npm run watch
```

### Frontend

```bash
cd arabic-sweets-frontend
npm install
npm run dev
```
